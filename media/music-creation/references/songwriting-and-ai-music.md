# Songwriting & AI Music — Complete Guide

Everything about songwriting craft, Suno AI prompt engineering, and parody adaptation.

## Song Structure

Common skeletons:
```
ABABCB  Verse/Chorus/Verse/Chorus/Bridge/Chorus  (most pop/rock)
AABA    Verse/Verse/Bridge/Verse (refrain-based)  (jazz, ballads)
ABAB    Verse/Chorus alternating                  (simple)
AAA     Strophic, no chorus                       (folk, storytelling)
```

Building blocks: Intro → Verse → Pre-Chorus → Chorus → Bridge → Outro

## Rhyme & Meter

- **Perfect**: lean/mean
- **Family**: crate/braid
- **Assonance**: had/glass (same vowels)
- **Consonance**: scene/when (different vowels, similar endings)
- **Slant**: suggests connection without locking down

**Internal rhyme**: "We pruned the lies from bleeding trees / Distilled the storm from entropy"

**Meter**: Match stressed syllables to beats. Say lyrics aloud — if you stumble, meter needs work.

## Emotional Arc

Map energy levels (1-10): Intro 2-3 → Verse 5-6 → Pre-Chorus 7 → Chorus 8-9 → Bridge varies → Final Chorus 9-10

**Core trick**: Contrast. Whisper before a scream. Sparse before dense. Silence is an instrument.

## Suno AI Prompts

**Style field formula:** Genre + Mood + Era + Instruments + Vocal Style + Production + Dynamics

```text
"Cinematic orchestral spy thriller, 1960s Cold War era, smoky sultry female vocalist,
big band jazz, brass section with trumpets and french horns, sweeping strings, minor key,
vintage analog warmth"
```

**Metatags in lyrics field:**
- Structure: `[Intro] [Verse] [Chorus] [Bridge] [Outro]`
- Vocals: `[Whispered] [Spoken Word] [Belted] [Falsetto] [Powerful] [Soulful]`
- Dynamics: `[High Energy] [Low Energy] [Explosive] [Orchestral swell]`
- Atmosphere: `[Melancholic] [Euphoric] [Nostalgic] [Intimate]`

## Parody Adaptation

1. Map syllables, rhyme scheme, stressed beats of original
2. Match stressed syllables to same beats
3. On held notes, match vowel sound of original
4. Keep some original lines intact for recognizability
5. Generate lots of material first, then fit best into structure

## Workflow

1. Write concept/hook first — emotional core
2. Map original structure if adapting
3. Generate raw material freely
4. Draft lyrics into structure
5. Read/sing aloud, fix meter
6. Build Suno style description
7. Add metatags to lyrics
8. Generate 3-5 variations minimum
9. Pick best, use Extend/Continue
10. Expect ~3-5 generations per good result
